
# <img width="20" height="20" alt="虚拟服务器存储申请" src="https://github.com/user-attachments/assets/36bd8274-1a7a-45d8-97f3-d99eeaf3c907" /> File Server文件服务器

这是一个基于 Go 语言开发的轻量级 HTTP 文件服务器。它摒弃了传统的固定密码认证，集成了 Google Authenticator (TOTP) 双重验证机制，为您的文件传输提供更高的安全性。

## 主要功能

- **🔒 动态安全认证**：使用标准 TOTP 协议，登录密码必须使用 Google Authenticator 生成的 6 位实时动态验证码。
- **📂 文件管理**：支持文件上传、下载及目录浏览。
- **智能下载**：自动设置 `Content-Disposition`，支持 `curl -OJ` 自动保存原始文件名。
- **🛡️ 安全防护**：内置路径穿越（Path Traversal）检测，防止访问非授权目录。
- **📝 双写日志**：访问记录同时输出到终端标准输出和 `logs/access.log` 文件，便于审计。

## 使用示例

该服务兼容标准的 HTTP Basic Auth，因此可以轻松通过浏览器或命令行工具访问：

### 下载文件：

```bash
# -u 指定用户名:动态验证码
# -OJ 使用服务器返回的文件名保存
curl -OJ -u admin:123456 http://IP/file.txt
```

### 上传文件：

```bash
curl -u admin:123456 -F "file=@local_file.txt" http://IP/upload
```
